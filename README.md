# 规则引擎 
#### 业务逻辑实现不再依赖于代码开发，可零代码实现复杂业务逻辑。

项目展示地址：http://ruleengine.cn/   
申请登录权限联系QQ：761945125  

### 技术栈/版本介绍：
- 所涉及的相关的技术有：
    - SpringBoot 2.1.2
    - RabbitMQ
    - Redis、Redisson
    - MySQL 5.7.9
    - Mybatis-plus
    - Mybatis-plus-generator
    - Maven 3.3
    - Swagger
    - alibaba OSS
